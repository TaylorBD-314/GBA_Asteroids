# GBA_Asteroids
A recreation of the GBA asteroids game written in C by Taylor Boese
==================================================================================================================================================================

To compile this code, have devkitPro installed, navigate to your Asteroids directory inside your command line, and use the command "make" to create a .gba file using the existing code.

Make sure you don't miss the already compiled .gba Asteroids file in this repository!

To easily play the .gba file provided in this repo on your PC, simply install a GBA emulator like mGBA, then run the file using your application.
